# AYUSH MEDICO – MEDICINE DELIVERY REQUEST SYSTEM (PRODUCTION READY)



## OBJECTIVE



Build a professional medicine delivery request system for Ayush Medico. This is NOT a normal e-commerce checkout.



The workflow must be:



Customer submits a medicine request → Pharmacist/Admin verifies the request → Admin confirms medicine availability and pricing → Customer pays → Order is prepared → Out for Delivery → Delivered.



The system must look modern, clean, mobile responsive, and suitable for a medical pharmacy.



---



# DELIVERY AVAILABILITY



Delivery is available ONLY within a 4 KM radius of Kurla West.



The customer must enter a complete delivery address.



Required fields:



• Full Name

• Mobile Number

• WhatsApp Number (checkbox: Same as Mobile)

• House / Building Name

• Street / Area

• Landmark

• Pincode

• Full Address

• Delivery Instructions (Optional)



The system should validate whether the address is within the supported delivery area.



If outside the service area display:



❌ Sorry, medicine delivery is currently available only within 4 KM of Kurla West.



Disable the submit button.



If inside:



✅ Delivery Available
